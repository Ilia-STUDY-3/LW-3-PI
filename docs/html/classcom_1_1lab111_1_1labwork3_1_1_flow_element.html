Подробное описание

Представляє абстрактний базовий інтерфейс (Component) для всіх елементів блок-схеми.

Цей клас оголошує операції, спільні для простих (Leaf) і композитних (Composite) об'єктів. Він є ключовим елементом патерну Composite, дозволяючи клієнтському коду працювати з усіма об'єктами уніфікованим способом.
