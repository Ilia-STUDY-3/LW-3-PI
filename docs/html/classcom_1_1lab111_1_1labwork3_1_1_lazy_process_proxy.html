Подробное описание

Клас, що виступає в ролі віртуального проксі для ProcessBlock.

Цей клас реалізує патерн Proxy, забезпечуючи "ліниву" (відкладену) ініціалізацію об'єкта ProcessBlock. Реальний об'єкт створюється лише тоді, коли клієнт вперше звертається до нього.
